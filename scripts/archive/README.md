## Why Archive Instead of Delete?

1. **Historical Reference:** Understanding past approaches
2. **Rollback Safety:** In case new versions have issues
3. **Learning:** New team members can see evolution
4. **Audit Trail:** Track what changed and when

- Scripts are archived when replaced by better versions
