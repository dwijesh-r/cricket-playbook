📊 Ranking Methodology

Batter Phase Rankings

SR percentile (40%) + Avg percentile (40%) + Boundary% percentile (20%). Target: 200 balls per phase. Sample-size weighted.

Bowler Phase Rankings

Economy percentile (50%) + Dot Ball% percentile (50%). Target: 120 balls per phase. Sample-size weighted.

Batter vs Bowling Type

SR (40%) + Avg (40%) + Survival Rate (20%). Min 50 balls. Target: 100 balls. Sample-size weighted.

Bowler vs Handedness

Economy percentile (50%) + SR percentile (50%). Min 50 balls. Target: 100 balls. Sample-size weighted.

Player Matchups

Head-to-head dominance: SR deviation (50%) + Avg deviation (30%) + Boundary% deviation (20%). Min 12 balls. Target: 50 balls.

Batter Composite

Career SR+Avg (30%) + Phase Performance (30%) + Boundary% (20%) + Average (10%) + Dot Ball Discipline (10%). Min 500 balls.

Bowler Composite

Career Econ+Avg (30%) + Phase Performance (30%) + Economy (20%) + Wicket-taking (10%) + Dot Ball% (10%). Min 300 balls.

Rankings are percentile-based with sample-size weighting. Full methodology: see config/thresholds.yaml > rankings.
